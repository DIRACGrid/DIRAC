========================
dirac-wms-job-reschedule
========================

Reschedule the given DIRAC job

Usage::

  dirac-wms-job-reschedule [option|cfgfile] ... JobID ...

Arguments::

  JobID:    DIRAC Job ID

Example::

  $ dirac-wms-job-reschedule 1
  Rescheduled job 1
