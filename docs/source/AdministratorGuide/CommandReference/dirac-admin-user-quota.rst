======================
dirac-admin-user-quota
======================

Show storage quotas for specified users or for all registered users if nobody is specified

Usage::

   dirac-admin-user-quota [user1 ...]

Example::

  $ dirac-admin-user-quota
  ------------------------------
  Username       |     Quota (GB)
  ------------------------------
  atsareg        |           None
  msapunov       |           None
  vhamar         |           None
  ------------------------------
