=========================
dirac-admin-get-site-mask
=========================

Get the list of sites enabled in the mask for job submission

Usage::

   dirac-admin-get-site-mask [options]

Example::

  $ dirac-admin-get-site-mask
  LCG.CGG.fr
  LCG.CPPM.fr
  LCG.GRIF.fr
  LCG.IBCP.fr
  LCG.IN2P3.fr
  LCG.IPNL.fr
  LCG.IPSL-IPGP.fr
  LCG.IRES.fr
  LCG.LAPP.fr
  LCG.LPSC.fr
  LCG.M3PEC.fr
  LCG.MSFG.fr
