======================
dirac-utils-file-adler
======================

Calculate alder32 of the supplied file

Usage::

  dirac-utils-file-adler [option|cfgfile] ... File ...

Arguments::

  File:     File Name

Example::

  $ dirac-utils-file-adler Example.tgz
  Example.tgz 88b4ca8b
